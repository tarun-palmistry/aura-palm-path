# AstraPalm (aura-palm-path)

Multilingual (English / Hindi) web app for AI-assisted palm reading, Vedic astrology, Kundali matching, horoscopes, and Razorpay unlocks. Frontend: React, Vite, Tailwind, Supabase client. Backend: Supabase Edge Functions (Deno) and PostgreSQL.

## Local development

```bash
npm install
npm run dev
```

Configure `VITE_SUPABASE_URL` and `VITE_SUPABASE_PUBLISHABLE_KEY` in your environment.
